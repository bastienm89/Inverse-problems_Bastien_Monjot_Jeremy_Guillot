# Inverse Problems - DIP-TV

Ce repository contient un notebook Python reproduisant partiellement les expériences de l'article :

"Deep Image Prior with TV Regularization".

## Objectif

L'objectif de ce projet est de comprendre et reproduire une partie des résultats de l'article en utilisant la bibliothèque DeepInverse.

## Contenu

Le notebook présente :

- le problème inverse considéré
- la méthode DIP-TV
- la nouveauté par rapport à l'état de l'art
- la compréhension des résultats de l'article
- une reproduction partielle des simulations de l'article

## Expérimentations

Les expériences sont réalisées sur plusieurs images avec différents niveaux de bruit.

Pour des raisons de temps de calcul et de ressources matérielles limitées, les simulations sont réalisées avec 1000 itérations et 3 images.
